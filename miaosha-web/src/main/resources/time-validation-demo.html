使用说明

API接口： GET /api/tickets/validateTime?date=2025-08-12

验证规则：

  • 只能抢购最近3天的票券（今天、明天、后天）
  • 不能购买过去的日期
  • 不能购买超过3天的未来日期
  • 日期格式必须为 yyyy-MM-dd

返回字段说明：

  • valid: 是否有效（true/false）
  • message: 验证结果消息
  • code: 状态代码
  • timeType: 时间类型（today/tomorrow/dayAfterTomorrow）
  • ticketDate: 票券日期
  • currentDate: 当前日期
